## 最短验收路径

```text
安装 Python 3.11 和 Node.js 20
  -> 复制 .env.example 为 .env 并生成 JWT 密钥
  -> 启动后端与前端
  -> 访问 /api/health
  -> 注册/登录并完成画像
  -> 进入智能对话，观察 Agent Timeline、证据和资源
  -> 查看知识库、学习画像、学习路径和复习反馈
  -> 访问 /api/code/status，确认当前代码执行模式
```

默认核心验收不需要 Docker，也不需要外部大模型 API Key。默认 `deterministic` 模式用于验证系统流程和界面；真实文本/视觉模型需要评委在设置页填写自己的兼容 Endpoint、Model 和 API Key。
