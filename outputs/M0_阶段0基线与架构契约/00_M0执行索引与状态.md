## 4. 冻结规则

1. 文档版本使用 `M0-v主版本.次版本`；当前为 `M0-v0.1-draft`。
2. 评测案例、知识点金标和画像数据使用不可变 ID；内容变化必须升版本，旧版本保留。
3. 指标结果必须绑定代码提交、数据版本、模型配置、运行时间和原始逐例结果。
4. 模拟画像和企业样例必须标注“合成/模拟”，不得改写为真实客户或真实用户。
5. `.env`、API 密钥、令牌、个人联系方式和原始敏感数据不得进入本目录。
6. 未运行的测试只能写“已收集”或“待运行”，不得写“通过”。
7. M0 变更需在 ADR 中写明原因、影响、回滚方式，并同步更新追踪矩阵。
