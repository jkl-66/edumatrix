## 6. 依赖和运行边界

- 后端依赖列在根目录 `requirements.txt`；前端依赖和脚本列在 `frontend/package.json`。
- `.env.example` 只作为键名/默认策略模板；真实 `.env`、令牌和密钥禁止进入文档和提交包。
- 本地非 Docker 路径是比赛首选：后端、前端、SQLite 和 deterministic fallback 可以独立启动；代码执行功能需按 `EDUMATRIX_SANDBOX_MODE` 的安全策略明确展示“可运行”或“拒绝执行”。
- 外部 LLM、Embedding、Playwright 浏览器和 Docker 都是环境条件，不能在没有安装/配置证据时写成评委必然可用。
