# RAG 核心知识点金标集

版本：`kp-rag-0.1`  
用途：覆盖率分母、画像诊断、资源生成、岗位任务和评测案例的共同知识点集合。正式发布前由领域评审人复核定义和先修关系。

## 1. 知识点分层

| 层级 | 计入官方覆盖率 | 说明 |
|---|---|---|
| 核心 `core` | 是 | 主领域最低可用能力，必须有讲解/实操/测量之一并有来源 |
| 扩展 `extended` | 否（单独报告） | 提升深度或迁移能力，不改变核心分母 |
| 先修 `prerequisite` | 否（单独报告） | 作为路径前置和画像差距，不与核心覆盖率混淆 |

## 4. 计入覆盖率的条件

一个知识点只有同时满足以下条件才记为“已覆盖”：

1. 产物或实操明确解释/执行了该知识点，而非仅出现关键词。
2. 至少一个来源引用或内部原创标记可追溯。
3. 至少一个题目、实操检查或人工评测能够区分“会/不会”。
4. 版本绑定到本金标集；无法判断的内容标为 `unknown`，不强行计入分子。

机器学习题库中的 25 个主题可作为先修/扩展材料，不能仅凭其文件数量将 KP-01 至 KP-32 宣称覆盖。
