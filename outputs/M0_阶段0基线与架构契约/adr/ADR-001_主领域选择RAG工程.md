# ADR-001：主领域选择 RAG 应用开发与幻觉评测

状态：`proposed`  
日期：2026-07-23

## 上下文

项目已有知识库、解析、检索、引用、辩论、代码实操、画像和评测基础，需要选择能形成岗位任务和客观技能证据的首个垂直领域。

## 决定

主领域固定为“RAG 应用开发、知识库构建与幻觉评测技能培训”，领域版本 `domain-rag-0.1`。

## 备选

- 通用机器学习课程：已有 25 章题库和 GitHub 素材，但难以直接体现知识生成、幻觉治理和岗位闭环。
- 通用 AI 学习助手：范围过宽，无法冻结知识点和评测分母。

## 影响和风险

可复用现有工程能力，但现有 25 章机器学习素材只能作为先修/扩展，必须补 RAG 专属课程。风险是领域专家评审不足；缓解为先冻结原创基准课和可追溯评测，正式结果前补人工评审。

## 撤销条件

若赛题官方书面要求限定其他领域，或主领域无法在当前系统完成课程摄入和黄金旅程，则新增 ADR 变更，不就地修改本记录。
