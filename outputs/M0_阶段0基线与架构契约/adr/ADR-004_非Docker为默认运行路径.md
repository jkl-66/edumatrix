# ADR-004：非 Docker 为比赛默认运行路径

状态：`proposed`  
日期：2026-07-23

## 决定

Windows 本地 Python/Node/SQLite 是比赛默认运行路径；Docker 是可选的代码隔离和部署增强，不作为核心功能唯一前提。

## 理由

评委环境可能没有 Docker，用户本机也明确希望降低安装复杂度。核心画像、知识库、聊天、路径和报告应在非 Docker 路径可验证。

## 安全边界

不可信代码不能为了演示直接在宿主运行。无安全沙箱时必须明确拒绝或使用经批准的受控本地研究模式；文档区分两者。

## 撤销条件

若赛方提供统一容器环境并明确要求 Docker，可将容器提升为推荐路径，但仍保留本地故障诊断和依赖说明。
