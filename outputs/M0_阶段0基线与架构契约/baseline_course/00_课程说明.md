# RAG 应用开发与幻觉评测实训课

课程 ID：`course-rag-engineering-m0`  
课程版本：`0.1.0`  
领域包：`domain-rag-0.1`  
内容性质：EduMatrix 项目组原创基准课程源文件，作为比赛演示输入；后续可在来源清单中补充经许可的外部材料。

## 课程目标

学习者完成本课程后，应能围绕一组公开或内部授权资料，完成“来源登记—结构化解析—分块与索引—检索调优—引用约束生成—幻觉评测—故障恢复”的最小 RAG 工程闭环，并能解释结果的证据边界。
