# EduMatrix 评委材料目录

本目录包含面向评委的最终技术说明、复现说明和验收证据。推荐直接使用：

- `EduMatrix_评委最终技术文档.docx` / `.pdf`：62 页排版后的主技术文档，含总体架构图、学习闭环图、Agent 协同图、RAG 隔离图、数据生命周期图、沙箱边界图和 10 张当前前端验收截图。
- `EduMatrix_A3_评委阅读版_方向A.pptx` / `.pdf`：20 页普通评委阅读材料，采用“深色技术·证据岛”方向，按官方 A3 评分口径组织。
- `render/viewer.html`：PPT 的零依赖翻页预览，旁边的 `slide01.png` 至 `slide20.png` 用于快速检查页面。
- `EduMatrix_评委最终介绍文档包_上杉绘梨衣.zip`：评委材料总包，已排除团队任务清单、整改记录、故障诊断笔记、私人联系方式、密码和 API Key。
- `00015352作品.zip`：赛方“作品安装/可执行文件”包，含脱敏演示数据库、前端构建产物、依赖清单和一键安装运行脚本。
- `00015352源码.zip`：赛方“作品源码”包，含后端、前端、测试和公开数据清单。
- `00015352介绍.zip`：赛方“介绍 PPT/演示视频和文档”包，当前已含 62 页 Word、PDF、20 页 PPT、PDF、技术图和项目截图；演示视频待队伍提供后重新生成。

压缩包内包含：

1. 评委阅读说明；
2. 完整 Word 主文档及 Markdown 总稿；
3. 系统设计、API 与数据字典、测试、部署和评委复现说明；
4. 需求—实现—测试—证据追踪矩阵与公开依据清单；
5. 20 页 PPT 及翻页预览；
6. 6 张技术图、10 张当前前端截图、脱敏验收报告及必要安全证据。

PPT 已纳入同一评委材料包；演示视频未纳入，因为队伍已确认官方提交要求未要求视频。最终文件命名和提交形式仍以官方系统最新通知为准。

最终材料的默认验收口径为 Windows 原生、无需 Docker、`deterministic` 模式；真实外部文本/视觉模型和 Docker 代码执行属于按环境配置的可选验收项。
