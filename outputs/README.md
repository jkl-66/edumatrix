# EduMatrix 文档交付目录

所有新文档基于 Git 基线 `2952dc1b17d793e5d76f54e1764348ebe50e4d5e`，并按照“已证实、部分实现、待验证、规划/建议”区分事实等级。

## 推荐阅读顺序

1. `EduMatrix_完整技术文档总稿.md`：最终主文档，覆盖需求分析、系统设计、Agent、RAG、算法、前端、数据库、测试、部署、安全、比赛对齐和答辩材料。
2. `EduMatrix_系统设计与实现方案.md`：开发与架构细节，包括异步边界、资源工厂、算法实现和验收标准。
3. `EduMatrix_API与数据字典.md`：API 路由、认证状态、请求响应和数据库字段。
4. `EduMatrix_需求实现测试追踪矩阵.md`：赛题要求到功能、代码、测试和证据的逐项追踪。
5. `EduMatrix_测试说明书.md`：测试分层、当前实际结果、比赛指标评测方法和待补用例。
6. `EduMatrix_部署运维说明书.md`：本地、Docker、配置、备份、监控和故障排查。
7. `EduMatrix_风险整改清单.md`：按严重程度排序的安全、工程和文档风险。
8. `EduMatrix_比赛答辩演示脚本.md`：七分钟演示流程、评委问答和失败预案。
9. `EduMatrix_软件杯赛题冲刺任务清单.md`：按当前 A3 软件杯赛题修订后的七项优先级、责任边界和提交清单。
10. `EduMatrix_评委环境安装与复现备忘录.md`：评委机器的 Python、Node、Docker、Playwright、环境变量、启动和验收步骤。
11. `EduMatrix_本轮整改记录_20260719.md`：本轮认证、RAG、沙箱、依赖整改及真实验证边界。
